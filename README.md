## 🚀 Stdio 模版快速配置指南

请严格按照以下步骤操作，确保一次配置成功：

### 1️⃣ 初始化设置
1. astrbot的webui界面，打开mcp服务器，点击界面右上角的 **「Stdio 模板」** 按钮。
2. **清空**或替换编辑框内的所有现有内容。

### 2️⃣ 填入配置文件
将下方代码块中的 JSON 内容完整复制并粘贴到编辑框中（此配置专用于适配当前环境）：

```json
{
  "command": "python",
  "args": [
    "脚本代码存放的绝对路径，如：/root/AstrBot/data/ssh_server.py"
  ],
  "env": {
    "SSH_HOST": "127.0.0.1",
    "SSH_PORT": "22",
    "SSH_USER": "root",
    "SSH_PASSWORD": "在这里填入你的真实密码",
    "PYTHONUNBUFFERED": "1"
  }
}

### 3️⃣ ⚠️ 关键检查点（保存前必读）
在点击保存前，**必须**手动修改以下两项：

> - 🔑 **修改密码**：找到 `"SSH_PASSWORD"` 字段，将 `"在这里填入你的真实密码"` 替换为你服务器的 **真实 SSH 密码**。
> - 🌐 **修改 IP**：
>   - 如果连接 **本机**：保持 `"127.0.0.1"` 不变。
>   - 如果连接 **远程服务器**：请修改为服务器的 **公网 IP**。

### 4️⃣ ✅ 验证与完成
1. 填好上述信息后，点击对话框右下角的 **「测试连接」** 按钮。
2. 观察反馈：
   - 如果提示 **“连接成功”** 或显示 **绿色对勾 ✅**，说明代码和配置完美！
   - 确认无误后，点击 **保存**。
