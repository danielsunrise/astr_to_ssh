请按照以下步骤操作，保证一次成功：

点击模版按钮：先点击右上角的 “Stdio 模板” 按钮。
清空或替换内容：将编辑框里的内容全部清空，然后填入下面这段 JSON（它是专门适配这个界面的格式）：
{
  "command": "python",
  "args": [
    "/root/AstrBot/data/ssh_server.py"
  ],
  "env": {
    "SSH_HOST": "127.0.0.1",
    "SSH_PORT": "22",
    "SSH_USER": "root",
    "SSH_PASSWORD": "在这里填入你的真实密码",
    "PYTHONUNBUFFERED": "1"
  }
}
关键检查点（请在点击“保存”前确认）：
修改密码：一定要把 "SSH_PASSWORD" 后面的文字改成你服务器的真实密码。
修改 IP：如果你要连接的是本机，用 127.0.0.1；如果是远程服务器，请填公网 IP。
测试连接：填好后，点击右下角的 “测试连接”。
如果提示 “连接成功” 或显示绿色的勾，说明代码和配置都完美！
